# NAVER Sync

This plugin synchronizes documents and files in Obsidian using Naver's memo service.

## Features

- Uses Naver's memo service as a backup storage for cross-device synchronization.
- Supports mobile devices.
- Syncs periodically.
- Syncs after a certain period of inactivity after the last edit.

## Usage

1. Create a folder in the [NAVER memo service](https://memo.naver.com/).
2. Install "NAVER Sync" from **Community Plugins** in Obsidian settings.
3. Log in to NAVER in the plugin settings.
4. Select "sync start" in the command palette.

## Privacy Protection

- This plugin does not collect any personal information.
- Your NAVER password is not stored by this plugin.

## Precautions

- Back up your files before using this plugin.
- Synchronization may fail due to unclear storage policies on NAVER's memo service.
- If a document conflict occurs, the most recently modified version will overwrite others. (Last Write Wins)
- Deletions made externally while the plugin is enabled will also be synchronized.
