# NAVER Sync

[English](./README.md) | 한국어

Obsidian의 문서와 파일을 NAVER의 메모 서비스를 이용해서 동기화하는 플러그인입니다.

## 기능

- NAVER의 메모 서비스를 백업 저장소로 사용하여 기기간 동기화
- 모바일 지원
- 일정 시간마다 동기화
- 최종 수정 후 일정 시간 수정이 없으면 동기화

## 사용법

1. [NAVER 메모 서비스](https://memo.naver.com/)에서 동기화용 폴더를 생성
2. Obsidian 설정에서 **커뮤니티 플러그인** 에서 "NAVER Sync" 설치
3. 플러그인 설정에서 NAVER 로그인
4. 명령어 팔레트에서 "sync start" 선택

## 개인정보 보호

- 이 플러그인은 어떠한 정보도 수집하지 않습니다
- NAVER 비밀번호를 저장하지 않습니다.

## 주의사항

- 사용전에 반드시 백업 하세요.
- NAVER의 메모와 동기화하는 것이 아닙니다.
- NAVER의 메모 서비스의 용량 정책이 명확하지 않아 동기화가 안될 수도 있습니다.
- 문서 충돌 발생시 마지막 수정된 문서로 덮어 씌워집니다. (Last Write Wins)
- 플러그인이 켜진 상테에서 vault 내용을 외부에서 삭제하더라도 그대로 삭제가 동기화가 되어 버립니다.
